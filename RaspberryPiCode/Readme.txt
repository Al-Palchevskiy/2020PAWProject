To get Raspberry Pi to compile and run the code
1. install Python3
2. download and install libraries for python3 by typing this into a terminal
sudo python3 -m pip install pybluez
sudo python3 -m pip install cobs
sudo python3 -m pip install protobuf
3. run the code by navigating to "masterControlTorque.py" directory
cd /home/pi/Desktop/2020PAWProject
4. compile and run the code
python3 masterControlTorque.py
5. to stop the code from running in the terminal, hold down Ctrl + C twice
